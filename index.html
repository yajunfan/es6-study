ES6学习

  • 1.let,var,const

    Ⅰ
    for(var i=0;i<3;i++){console.log("内部i",i);}
    console.log("外部i",i);
    打印的结果为
    内部i 0，内部i 1，内部i 2， 外部i 3
    ===》
    Ⅱ
    for(let j=0;ji<3;j++){ console.log("内部j",j);}
    console.log("外部j",j);
    打印的结果为
    内部j 0,内部j 1, 内部j 2
    Uncaught ReferenceError: j is not defined
    ===》
    Ⅲ
    得出结果:
    1) 在for循环中的变量，通过var定义的是全局变量，通过let定义的为私有变量
    2) 在函数内部定义的变量，不管是let还是var，都是私有变量，外部无法获取
    Ⅰ
    console.log(a)；//undefined
    var a = 2;
    var a = 3;
    console.log(a) //3
    ===》
    Ⅱ
    console.log(b)；Uncaught SyntaxError: Identifier 'b' has already been declared
    let b = 2;
    let b = 3;
    console.log(b) Uncaught SyntaxError: Identifier 'b' has already been declared
    ===》
    Ⅲ
    得出结果:
    1) var定义可以多次定义，let定义之后，再次定义会报错
    2) 在定义之前打印a变量，为undefined，表示a变量名提升，只是未赋值 而let定义赋值之前打印b，报错，表示let不会变量名提升

  • 2.变量的解构赋值

    Ⅰ
    针对【 数组 】的形式
    let [a,b,c,d,[e,f]]=[1,2,3,4,[5,6]];
    let ary = [a,b,c,d,[e,f]];
    ary[1] = 6;
    console.log(a,e); // 1,5
    console.log(ary) // [1,2,3,4,[5,6]]
    ===》
    Ⅱ
    针对【 对象 】的形式
    for(let j=0;ji<3;j++){ console.log("内部j",j);}
    console.log("外部j",j);
    打印的结果为
    内部j 0,内部j 1, 内部j 2
    Uncaught ReferenceError: j is not defined
    ===》
    Ⅲ
    得出结果:
    1) 在for循环中的变量，通过var定义的是全局变量，通过let定义的为私有变量
    2) 在函数内部定义的变量，不管是let还是var，都是私有变量，外部无法获取

    3）可以使用concat(),slice()的现有方法
    4）可以进行遍历取值
    ===》
    Ⅱ
    针对【 对象 】的形式
    1）可以使用Object.assign()
    2) 可以使用JSON.parse(JSON.stringify())
    ===》
    Ⅲ
    得出结果:
    1) 对于对象和数组，使用（2）(4)的方法是深克隆
    2) 其他两种方式，当为数组时，只有数组时纯数组，一维的数组，可以实现深克隆，对于有更深层的数组，不能达到深克隆的效果
    3）jquery中的$.extend是深克隆
